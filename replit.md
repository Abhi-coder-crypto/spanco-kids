# Spanco Kids Fashion

## Overview
A premium kids fashion e-commerce website built with React, Vite, and Tailwind CSS. Features product categories for infants, boys, girls, ethnic wear, western wear, best sellers, sale items, and festive collections.

## Tech Stack
- **Frontend**: React 18, TypeScript, Vite 5
- **Styling**: Tailwind CSS, shadcn/ui components
- **Routing**: React Router DOM v6
- **State Management**: React Context (StoreContext), TanStack React Query
- **Animations**: Framer Motion
- **Charts**: Recharts

## Project Structure
- `src/` - Main source directory
  - `components/` - Reusable UI components
  - `pages/` - Page-level components
  - `context/` - React context providers (StoreContext)
  - `data/` - Static data files
  - `hooks/` - Custom React hooks
  - `lib/` - Utility functions
  - `assets/` - Static assets (images, etc.)
- `public/` - Public static files

## Running the Project
- Development: `npm run dev` (runs Vite dev server on port 5000)
- Build: `npm run build`
- Preview: `npm run preview`

## Deployment
- Static deployment with `dist` as the public directory
- Build command: `npm run build`
